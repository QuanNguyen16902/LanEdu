# 🎯 UI Style Guide – Minimal Professional (Gold Theme)

## 1. 🎨 Color System

### Primary (Gold)
- Primary: #C9A227
- Hover: #A8841F
- Light: #F5E7B2

### Secondary (Teal)
- Secondary: #0F766E
- Hover: #115E59
- Light: #CCFBF1

### Neutral Colors
- Background: #FFFFFF
- Surface: #F9FAFB
- Border: #E5E7EB

### Text Colors
- Primary Text: #111827
- Secondary Text: #6B7280
- Disabled: #9CA3AF

### Semantic
- Success: #16A34A
- Error: #DC2626

---

## 2. 🔤 Typography

**Font Family**
- Inter / SF Pro / Roboto

**Font Size & Weight**
- Heading 1: 20px / Medium / line-height 28
- Heading 2: 18px / Medium / line-height 26
- Body: 14–16px / Regular / line-height 22–24
- Caption: 12–13px / Regular / line-height 18

**Rules**
- Max 3 font sizes per screen
- Avoid overusing bold
- Prioritize readability

---

## 3. 📏 Spacing System (8pt Grid)

- 4px  → very tight spacing  
- 8px  → small gap  
- 12px → default spacing  
- 16px → component padding  
- 20px → small section  
- 24px → section spacing  
- 32px → large layout spacing  

**Rules**
- Always use multiples of 8
- Avoid random spacing values

---

## 4. 📦 Component Style

### Border
- 1px solid #E5E7EB

### Radius
- 6px – 10px (max 12px)

### Shadow
- 0 1px 2px rgba(0,0,0,0.05)

**Rules**
- Keep components lightweight
- Avoid heavy shadows
- Avoid overly rounded corners

---

## 5. 🔘 Button System

### Primary Button
- Background: Gold (#C9A227)
- Text: White
- Padding: 10px 16px
- Radius: 8px

### Secondary Button
- Background: Transparent
- Border: 1px solid #E5E7EB
- Text: #111827

**Rules**
- Only 1 primary button per dialog
- Secondary actions must be less prominent

---

## 6. 🧠 Interaction Rules

- Prioritize speed and clarity
- Max 2–3 actions per screen
- Always include states:
  - Default
  - Hover
  - Active
  - Disabled

---

## 7. 📋 Dialog (Attendance)

### Layout
- Width: 360–420px
- Padding: 16–20px
- Spacing between items: 8–12px

### Structure
[Title]

[List content / students]

[Cancel]   [Confirm / Check-in]

### Rules
- Optimize for fast interaction (< 3 seconds)
- Avoid unnecessary visual elements
- Focus on readability and scanning

---

## 8. 🚫 Anti-AI Design Rules

Avoid:
- Gradients
- Bright/yellow overuse
- Heavy shadows
- Large radius (>12px)
- Perfect symmetry layouts

Prefer:
- Natural spacing
- Subtle alignment differences
- Realistic product feel

---

## 9. 🎯 Color Usage Ratio

- 80% Neutral (white, gray)
- 15% Secondary (teal)
- 5% Primary (gold)

---

## 10. 🧩 Design Principles

- Clarity > Aesthetics  
- Speed > Decoration  
- Consistency > Creativity  

---

## 11. 💡 Notes

- Gold is used as an accent, not background
- Keep UI serious and professional
- Design for daily usage (efficient, predictable)
